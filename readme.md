## Características

- Desplazamiento suave entre secciones con Hammer.js
- Animaciones con Lottie
- Diseño responsivo con Bootstrap
- Efectos de partículas con tsParticles
- Monitorización del rendimiento con stats.js
- Scripts personalizados para la carga (loader.js), la animación de escritura (typing.js), el desplazamiento (scroll.js), el desenfoque (desenfoque.js) y las animaciones de movimiento (lottieLoader.js)
